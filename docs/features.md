# Features

Minimal personal one-pager.

## Page anatomy

1. **Header** — portrait, name, bio, LinkedIn / Telegram links (Tabler icons left of labels)
2. **Projects** — horizontal scroll gallery backed by a content collection
3. **Timeline** — career history stored as inline data in `index.astro`

## Feature summary

- Light and dark color schemes based on system preference
- Local, optimized images with uncropped project media
- Fluid full-bleed project gallery
- Mouse-tracking tilt on project cards and portrait
- Data-driven project content
- Contact-link obfuscation and crawler preferences
